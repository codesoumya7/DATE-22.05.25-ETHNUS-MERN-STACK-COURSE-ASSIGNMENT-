Introduction to HTML

HTML (Hypertext Markup Language) is used to create documents on the World Wide Web. It is simply a collection of certain key words called "Tags" that are helpful in writing the document to be displayed using a browser on the Internet. HTML was developed by Tim Berners-Lee in 1992.
